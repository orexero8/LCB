**PGlite** runs PostgreSQL in-process with `--max-connections 10`. Prisma ORM (runtime only) wraps `pg.Client` via `@prisma/adapter-pg`. The Prisma CLI binary cannot connect to PGlite вҖ” all schema/seed operations use raw `pg.Client`.
